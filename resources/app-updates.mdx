## v1.2.0 (2025-06-20)
- Added support for cross-chain transfers with lower fees.
- Improved onboarding flow for new users.
- Enhanced security for wallet recovery.
- Fixed minor UI bugs in the trading dashboard.

## v1.0.0 (2025-04-01)
- Initial launch of Dexari platform.
- Spot trading, wallet management, and rewards features available.
